Carte des centrales nucléaires à travers le monde

Retour au projet

Cette carte représente l'emplacement des centrales nucléaires référencées dans les bases de données OpenStreetMap.

  • Ces données sont extraites par le biais d'un script python et d'une interrogation via le service XAPI.
  • Les icones représentent chaque centrale, les icones jaune et noir les centrales actives, les icones grises les centrales arrêtées.
  • Autour de chaque centrale un cercle de couleur (dégradé du rouge au jaune) marque une zone de 100 km autour de la centrale.

Extraction des données.

Les données représentées sont extraites de la géolocalisation réalisée par les contributeurs du projet OpenStreetMap.
Si vous remarquez des erreurs, vous pouvez contribuer au projet OpenStreetMap et ajouter, modifier les données des centrales nucléaires.

Dans OpenStreepMap chaque élément est marqué par des couples clés-valeurs (clé=valeur) permettant de spécifier l'objet géographique.

Ainsi les centrales électriques sont repérés par :

  • power=generator

Si cette centrale est nucléaire sont ajoutés :

  • generator:source = nuclear (ou anciennement power_source = nuclear)

Optionnellement des données complémentaires peuvent être spécifiée (et géré par la carte) :

  • ref = *
  • name = *
  • operator = *
  • generator:source = * ou anciennement power:source = *
  • generator:output = * ou generator:output:electricity = * ou anciennement power_rating = *
  • url = *
  • wikipedia = *
  • start_date = *
  • end_date = *
  • disused = yes

Exemple :

  • power=generator
  • name = Centrale nucléaire du Blayais
  • operator = EDF
  • generator:source = nuclear
  • generator:output = 3600 MW
  • wikipedia = http://fr.wikipedia.org/wiki/Centrale_nucléaire_du_Blayais
  • start_date = 1981
