Cette carte représente l'emplacement des installations nucléaires référencées dans la base de données géographique OpenStreetMap.

  • Ces données sont extraites par le biais d'un script python et d'une interrogation via le service Overpass API.
  • Les icones représentent chaque installation, les icones colorés les installations actives, les icones grises les installations arrêtées (mais sous surveillence).
  • Autour de chaque centrale nucléaire un cercle de couleur (dégradé du rouge au jaune) marque une zone de 100 km, potentiellement dangereuse en cas d'accident grave.
  • Sont représentés :
    • Les mines d'uranium
    • Les usines de traitement et d'enrichissemnt de l'uranium
    • Les centrales nucléaires
    • Les centres de retraitement et de stockage des déchets radioactifs
    • Les sites d'explosions nucléaires

Extraction des données.

Les données représentées sont extraites de la géolocalisation réalisée par les contributeurs du projet OpenStreetMap.
Si vous remarquez des erreurs de données, vous pouvez contribuer au projet OpenStreetMap et ainsi ajouter, modifier les données des installations nucléaires et de tout autres objets géographiques.

Dans OpenStreepMap chaque élément est marqué par des couples clés-valeurs (clé=valeur) permettant de spécifier l'objet géographique.

Ainsi les centrales électriques sont repérés par :

  • power=generator

Si cette centrale est nucléaire est ajoutés :

  • generator:source = nuclear (ou anciennement power_source = nuclear)

Optionnellement des données complémentaires peuvent être spécifiée (et géré par la carte) :

  • ref = *
  • name = *
  • operator = *
  • generator:source = * ou anciennement power:source = *
  • generator:output = * ou generator:output:electricity = * ou anciennement power_rating = *
  • url = *
  • wikipedia = *
  • start_date = *
  • end_date = *
  • disused = yes

Exemple :

  • power=generator
  • name = Centrale nucléaire du Blayais
  • operator = EDF
  • generator:source = nuclear
  • generator:output = 3600 MW
  • wikipedia = http://fr.wikipedia.org/wiki/Centrale_nucléaire_du_Blayais
  • start_date = 1981
